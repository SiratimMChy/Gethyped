# GetHyped - High-Performance Social Content Agency Website
**Rebuilt by Siratim Mustakim Chowdhury**  
*Development Cycle: April 14 – April 20, 2026*

---

## ⚡ Project Overview
A high-fidelity React application developed for **GetHyped**, a social-first content agency. This project demonstrates the ability to implement advanced UI/UX patterns and complex scroll-based animations within a high-intensity development sprint from **April 14th** to its final production push on **April 20th, 2026**.

---

## 🛠 Tech Stack
- **Framework**: React 19.2.4 (Vite)
- **Animation Suite**: GSAP 3.15.0 (ScrollTrigger) & Framer Motion 12.38.0
- **Styling**: Tailwind CSS 4.2.2
- **Icons**: React Icons 5.6.0
- **Performance**: Optimized for 60FPS hardware-accelerated motion.

---

## 🎭 Key Features & Implemented Logic

### 📍 Advanced Stacking Engine (GSAP)
- **Expertise Architecture**: Developed a synchronized cross-card stacking system where elements transition vertically based on scroll depth, utilizing precision GSAP timelines.

### 📍 High-Fidelity Interactivity
- **Selected Work Portfolio**: Implemented dynamic Y-axis parallax effects and interactive "Jelly" buttons with elastic spring physics.
- **3D Interactive Layouts**: Integrated responsive 3D tilt-on-hover logic for project cards to enhance user engagement.

### 📍 Modern Development Standards
- **Responsive Fluidity**: Engineered a unified mobile and desktop layout system that ensures animation integrity across all screen sizes.
- **Efficient Rendering**: Intelligent video management ensuring smooth playback without taxing system resources.

---

## 📂 Project Structure
- `Expertises`: High-end GSAP animated card stacking.
- `SelectedWork`: Parallax scroll showcases and interactive components.
- `Hero & Intro`: High-impact visual layers with entry animations.
- `Navbar`: Smart-sticky navigation with responsive logic.
