# GetHyped - Social-First Content Agency Website

A modern, interactive website for GetHyped, a social-first content agency specializing in short-form video content and strategic social media solutions.

## 🚀 Features

- **Modern React Architecture**: Built with React 19 and Vite for optimal performance
- **Smooth Animations**: Advanced scroll-based animations using Framer Motion and GSAP
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Interactive Components**: Engaging user interface with hover effects and transitions
- **Performance Optimized**: Fast loading times and smooth scrolling experience

## 🛠️ Tech Stack

- **Frontend Framework**: React 19.2.4
- **Build Tool**: Vite 8.0.4
- **Styling**: Tailwind CSS 4.2.2
- **Animations**: 
  - Framer Motion 12.38.0
  - GSAP 3.15.0
- **Icons**: React Icons 5.6.0
- **Code Quality**: ESLint with React-specific rules

## 🚀 Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd gethyped
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`

### Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint for code quality checks
